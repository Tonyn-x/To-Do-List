# 📝 App Lista To-Do (Gerenciador de Tarefas)

Projeto prático de desenvolvimento web focado na criação de um gerenciador de tarefas dinâmico e responsivo, desenvolvido para o curso de Análise e Desenvolvimento de Sistemas da Uninter.

🔗 **Link do projeto online:** https://tonyn-x.github.io/To-Do-List/
---

## 🎯 Visão Geral do Sistema
O aplicativo To-Do List foi projetado para organizar tarefas diárias com foco em usabilidade e design moderno. O sistema permite que o usuário cadastre tarefas detalhadas, defina níveis de prioridade e busque rapidamente por atividades específicas através de uma barra de pesquisa dinâmica.

## 💻 Tecnologias Utilizadas
- **HTML5:** Estruturação semântica em blocos e formulários.
- **CSS3:** Interface moderna estilo "Dashboard", com design 100% responsivo utilizando Flexbox e CSS Grid (adaptação inteligente de layout entre Desktop e Mobile).
- **JavaScript:** Manipulação do DOM para criação, leitura, atualização e exclusão (CRUD) de elementos na tela.

## ⚙️ Funcionalidades Implementadas
- **Cadastro Completo:** Criação de tarefas exigindo Título, Descrição e nível de Prioridade (Baixa, Média, Alta).
- **Barra de Pesquisa:** Filtro dinâmico em tempo real que localiza tarefas pelo título digitado.
- **Gestão de Status:** - **Concluir:** Altera o visual da tarefa (fundo verde e texto tachado) para indicar finalização.
  - **Editar:** Permite alterar os dados da tarefa já criada.
  - **Excluir:** Remove a tarefa da lista.
- **Indicadores Visuais:** Tags de cores dinâmicas para as prioridades (Azul para Baixa, Laranja para Média, Vermelho para Alta).
- **Feedback Visual (UX):** Mensagem de "Nenhuma tarefa cadastrada" quando a lista está vazia e efeitos de `hover` nos botões de ação.

1. Clone este repositório ou baixe o arquivo ZIP.
2. Extraia os arquivos no seu computador.
3. Dê um duplo clique no arquivo `index.html` para abrir diretamente em qualquer navegador web.
